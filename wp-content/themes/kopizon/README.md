# Kopizon Premium WordPress Theme

A premium, high-performance WordPress theme designed for the **Enn Rev Enn Sourir** foundation. This theme is built with a focus on charity, transparency, and user engagement, featuring deep integration with WooCommerce and support for Elementor.

## Features

- **Modern & Responsive Design**: Custom-built using Vanilla CSS and Bootstrap 4.
- **Dynamic Content**: Powered by WordPress templates and custom functions.
- **WooCommerce Ready**: Optimized for donations and merchandise.
- **Elementor Compatible**: Support for drag-and-drop page building.
- **Impact Stats**: Dynamic counter for showcasing foundation achievements.
- **Custom Navigation**: Built-in mega-menus and mobile-optimized drawers.

## Installation

1. Login to your WordPress admin dashboard.
2. Navigate to **Appearance > Themes**.
3. Click **Add New** and then **Upload Theme**.
4. Select the `kopizon.zip` file and click **Install Now**.
5. After installation, click **Activate**.

## Dependencies

For full functionality, the following plugins are recommended:

- **WooCommerce**: For managing donations and products.
- **GiveWP**: For advanced donation forms.
- **Contact Form 7**: For contact and volunteer forms.
- **Elementor**: For custom page layouts.
